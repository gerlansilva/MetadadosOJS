# Deploy no Streamlit Community Cloud

Envie o conteúdo desta pasta para a raiz de um repositório GitHub chamado, por exemplo, `metaojs`.

A raiz do repositório deve mostrar diretamente:

- `app.py`
- `metaojs_core.py`
- `requirements.txt`
- `assets/`
- `.streamlit/`

No Streamlit Community Cloud use:

- Branch: `main`
- Main file path: `app.py`
- Python: 3.12

Evite nomes de pasta com espaços ou acentos no caminho do aplicativo.
